# Contexto
Quero criar uma landing page moderna, premium e altamente conversiva para uma barbearia. O objetivo principal é gerar agendamentos via WhatsApp.

# Tarefa
Crie uma landing page completa, responsiva e mobile-first para uma barbearia premium.

# Estilo visual
- Visual sofisticado, masculino e elegante
- Paleta escura: preto, grafite, branco e dourado/cobre
- Tipografia premium
- Fotos grandes, cards elegantes, sombras suaves e microanimações
- Design moderno, limpo e com forte foco em conversão

# Estrutura da página
1. Hero
   - Headline forte e premium
   - Subheadline com proposta de valor
   - Botão principal: “Agendar pelo WhatsApp”
   - Exibir avaliação do Google: estrelas, nota e texto como “4.9 no Google”
   - Imagem ou background de barbearia premium

2. Agendamento via WhatsApp
   - CTA fixo ou destacado
   - Botão deve abrir WhatsApp com mensagem pronta:
   “Olá! Gostaria de agendar um horário na barbearia.”
   - Use um número placeholder: +55 11 99999-9999

3. Problema
   - Mostrar dores do cliente: corte mal feito, espera longa, ambiente comum, falta de atenção aos detalhes

4. Solução
   - Apresentar a barbearia como solução: atendimento premium, profissionais experientes, ambiente confortável, acabamento impecável

5. Galeria
   - Grid responsivo com imagens de cortes, barba, ambiente e detalhes
   - Use imagens placeholder elegantes por enquanto

6. Localização
   - Seção com endereço placeholder
   - Mapa incorporado ou card visual de localização
   - Botão “Como chegar”

# Requisitos
- Landing page em uma única página
- Totalmente responsiva em mobile, tablet e desktop
- Usar Tailwind CSS e componentes modernos
- Não criar backend
- Não criar login
- Foco em performance, SEO básico e conversão
- Incluir CTAs repetidos ao longo da página

# Resultado esperado
Entregue uma landing page pronta, visualmente premium, com copy persuasiva em português brasileiro e experiência mobile-first.
